#Port Scanner

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.8+](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)

Ethical multi-threaded TCP port scanner for cybersecurity education. Detects open ports on authorized targets (own VMs/hacklabs). Part of Nemesis ethical hacking toolkit.

## Features
- Scans ports 1-1024 (common services)
- Multi-threaded (200 workers) for speed
- Cross-platform: Windows/Linux/macOS
- No external dependencies
